About the Project:

Technology: 
Developed and implemented a full-stack video website using React and Redux for front-end, Node.js for back-end API support, MongoDB as the database, and Firebase for cloud storage. 

Structure: 
Implemented a fully functional website, featuring user registration, search functionality, and comments. Streamlined publisher experience with features like content uploading, deletion, upgrades, and recommendations.

Design: 
Prioritize the creation of an interactive user experience (UX) and user interface (UI) design that caters to both user and business perspectives.
